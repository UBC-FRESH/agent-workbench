Roadmap Overview
==============

Agent Workbench follows a phase/task/subtask workflow documented in `ROADMAP.md`. Below is a summary of completed and active phases. For the complete issue tracker map with parent issues, branches, and status, see `ROADMAP.md` in the repository root.

**Phase Progression**: P0–P100 cover governance scaffold, worker protocol templates, SDK probes, workflow graphs, model comparison, delegation economics, and documentation preparation. Each phase has a GitHub parent issue, feature branch, and child tasks.

**Current Active Phase**: P101 — Sphinx technical documentation with GitHub Pages deployment (this page). This phase creates the published docs infrastructure for public alpha release.

**Upcoming Phases**:
- P95: Retrieval and modelling-agent usability
- P98: Reporting-worker template packaging
- P99: Economics dashboard and release criteria
- P100: Public alpha readiness review
- P101: Sphinx docs (in progress)

The full roadmap with all phase details, child issues, and closeout verification is maintained in `ROADMAP.md`.
