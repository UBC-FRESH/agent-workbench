VS Code Chat Bridge Playbook
============================

This guide covers using Agent Workbench with the VS Code Chat bridge for coordinated
supervisor-worker agent interactions.

> **Status**: This guide is in progress. See [AGENTS.md](https://github.com/UBC-FRESH/agent-workbench/blob/main/AGENTS.md) for the full delegation contract and file-based handoff protocol.

The VS Code Chat bridge enables the local supervisor to delegate bounded tickets to worker agents running inside Copilot Chat sessions, capturing evidence via session archives and result files in `runtime/`.
