Agent Workbench
===============

Agent Workbench is a local supervisor tooling sandbox for supervised multi-agent
development workflows. It provides reusable workflow graph templates, governance
protocols, evidence validation contracts, and CLI tooling to coordinate
supervisor-worker agent pairs executing bounded roadmap tasks.

The project is currently in public-alpha preparation. Documentation covers
governance rules, playbooks for common workflows, template catalogs, protocol
specifications, and implementation guidance for target-project integration.
